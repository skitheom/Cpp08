## Exercises
### Ex00: Easy find
- STLを用いて指定された値をリスト内に探すtemplate関数を作成
- `T::iterator it`: データ型の操作が簡便になる
- `std::find` / `std::find_if` : 手動探索は不要
- associative containers 対応不要
- 値が存在しない場合にexceptionまたはエラー値
- const考慮 `const_iterator`

### Ex01: Span
- 最大N個のint型数値を保存できるSpanクラス
- overflowに注意
- spanとは2点間の差（絶対値）
